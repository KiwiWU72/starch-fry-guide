# starch-fry-guide
🚀 油炸粉技術全解 — Kiwi R&D 數位教材
這是一個基於 HTML5 + SVG 動畫 的互動式食品科學教材，專為 B2B 技術業務與食品研發設計。本專案將複雜的高分子化學（澱粉、蛋白質）轉化為直觀的視覺化模型，協助解決油炸食品常見的「脫殼」、「回軟」與「上色不均」等技術痛點。

🛠 近期重大更新 (2026-05)
本版本進行了底層邏輯的深度擴充，從單一的澱粉機制進展到「澱粉-蛋白質-糖類」三位一體的複配體系：

1. 蛋白質系統與介面科學 (Protein-Interface Dynamics)
肌球蛋白溫度響應模型：新增 SVG 互動動畫，模擬肌球蛋白（Myosin）在 25°C 至 88°C 的構象變化（頭部展開、柄區熔解、S-S 鍵交聯）。

肉品前處理第一性原理：導入「鹽溶性蛋白萃取」理論，解釋 NaCl 與磷酸鹽如何透過離子強度調節溶出肌球蛋白，形成天然介面黏合劑。

熱誘導交聯機制：闡述麵衣蛋白與肉體游離蛋白在 180°C 下形成的共價雙硫鍵縫合效應。

2. 還原糖梅納反應矩陣 (Reducing Sugar Matrix)
反應活性排序：對比木糖、果糖、葡萄糖、麥芽糖、乳糖、DE 10/18 糊精及蔗糖。

DE 值（Dextrose Equivalent）物理意義：深入探討還原端密度如何決定梅納反應（Maillard Reaction）的上色曲線與吸濕性（Tg 玻璃態轉變）。

蔗糖上色異常解釋：詳解非還原糖在缺乏「梅納啟動子」時，導致顏色死白與瞬間焦化的物理原因。

3. 老化動力學與雙面刃效應 (Retrogradation Paradox)
含水量與質感對照：新增高/低含水量體系下老化的差異分析。

雙炸法 (Double Frying) 科學化：解釋如何利用「有意的人為老化」在低水分環境下建立硬脆的結晶骨架。

支鏈/直鏈澱粉 Cluster Model：優化樹狀圖動畫，精確區分短期凍融析水（直鏈為主）與長期硬化（支鏈短鏈為主）的分子路徑。

🔬 技術核心 (The First Principles)
澱粉修飾：以「打鋼釘（磷酸交聯）」與「套手套（醋酸取代）」形象化比喻共價鍵穩定性與立體障礙效應。

膨發系統：視覺化 SAPP 與小蘇打的「雙作用釋氣曲線」，強調慢釋出對氣孔結構的重要性。

介面化學：解析蛋白質兩親性分子（Amphiphilic）如何降低澱粉漿與肉類油脂間的介面張力。

📊 應用場景
B2B 業務提案：透過互動動畫快速與客戶研發對接，建立專業技術壁壘。

客訴診斷：利用「業務口袋題」快速判斷生產線上脫殼或吸油率過高的底層成因。

新進人員培訓：食品科學基礎知識的降維教學。
